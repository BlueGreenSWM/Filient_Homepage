# 액션 유형

Filient는 조건에 맞는 파일들에 대해 다양한 자동화 액션을 실행할 수 있습니다.

## 파일 이동 액션

### 폴더로 이동
파일을 지정된 위치로 이동:
- **대상 폴더 선택**: 이동할 폴더 지정
- **하위 폴더 생성**: 날짜/유형별 자동 생성
- **덮어쓰기 옵션**: 중복 시 처리 방법
- **원본 삭제**: 이동 후 원본 처리

```
예시:
- 다운로드 → 문서 폴더
- 사진 → 연도/월별 폴더
- 임시 파일 → 휴지통
```

### 계층 구조 정리
폴더 구조 자동 생성:
- **날짜별**: 2024/01/15 형식
- **유형별**: 문서/이미지/동영상
- **프로젝트별**: 프로젝트명/하위구조
- **사용자 정의**: 커스텀 규칙

## 파일명 변경 액션

### 이름 바꾸기
파일명 수정 옵션:
- **접두사 추가**: 파일명 앞에 추가
- **접미사 추가**: 파일명 뒤에 추가
- **문자 치환**: 특정 문자 변경
- **대소문자 변경**: 대문자/소문자 변환

### 순차 번호
일련번호 추가:
- **숫자 형식**: 001, 002, 003...
- **시작 번호**: 임의 번호부터 시작
- **자릿수**: 2자리, 3자리, 4자리
- **구분자**: _, -, 공백 등

### 날짜 추가
타임스탬프 삽입:
- **현재 날짜**: 오늘 날짜 추가
- **생성 날짜**: 파일 생성일 사용
- **수정 날짜**: 최종 수정일 사용
- **형식 설정**: YYYY-MM-DD, YYYYMMDD 등

## 파일 처리 액션

### 압축
파일 압축 처리:
- **ZIP 생성**: 표준 ZIP 형식
- **압축률 설정**: 최대/표준/빠름
- **분할 압축**: 크기 제한 설정
- **암호화**: 비밀번호 보호

### 압축 해제
압축 파일 추출:
- **대상 폴더**: 추출 위치 지정
- **폴더 생성**: 압축명으로 폴더 생성
- **덮어쓰기**: 중복 파일 처리
- **압축 파일 삭제**: 해제 후 삭제

### 변환
파일 형식 변환:
- **이미지 변환**: JPG → PNG, HEIC → JPG
- **문서 변환**: DOCX → PDF
- **비디오 변환**: MOV → MP4
- **품질 설정**: 변환 품질 조절

## 조건부 액션

### IF-THEN 규칙
조건별 다른 액션:
```
IF 파일 크기 > 100MB
  THEN 압축 후 이동
ELSE IF 파일 크기 > 10MB
  THEN 그대로 이동
ELSE
  THEN 복사
```

### 체인 액션
연속 작업:
1. 이름 변경
2. 폴더로 이동
3. 태그 추가
4. 알림 발송

### 예외 처리
오류 시 대체 액션:
- **재시도**: 실패 시 재시도
- **대체 액션**: 다른 액션 실행
- **건너뛰기**: 다음 파일 처리
- **중단**: 전체 작업 중단
