# 조건 유형

Filient는 다양한 조건을 조합하여 정교한 파일 관리 규칙을 만들 수 있습니다.

## 파일 속성 조건

### 파일 이름
파일명 기반 필터링:
- **포함**: 특정 문자열 포함
- **일치**: 특정 이름과 완전히 일치
- **시작**: 특정 문자열로 시작
- **끝**: 특정 문자열로 끝남
- **정규식**: 정규식 패턴 매칭

```
예시:
- "IMG_"로 시작하는 파일
- ".tmp"로 끝나는 파일
- "2024"를 포함하는 파일
```

### 파일 확장자
파일 형식별 분류:
- **확장자 직접 입력**: .jpg, .pdf, .docx 등 쉼표(,)로 구분
- **확장자 그룹**: 이미지, 문서, 동영상 등

추가 이후 각 확장자를 삭제를 통한 커스터마이징도 지원합니다.

```
확장자 그룹:
- 이미지: jpg, jpeg, png, gif, bmp, heic, svg, webp, ico, tiff
- 문서: pdf, docx, doc, hwp, txt, md, xlsx, xls, pptx, ppt
- 오디오: mp3, wav, aac, m4a, flac, wma, ogg, aiff
- 비디오: mp4, avi, mkv, mov, wmv, flv, webm, m4v

이외에도 압축, 코드, 실행 등의 그룹 존재
```

### 파일 크기
크기 기반 필터:
- **크다**: 파일 크기가 입력값 초과
- **작다**: 파일 크기가 입력값 미만
- **같다**: 파일 크기가 입력값과 동일
- **사이**: 파일 크기가 a 이상 b 이하

```
단위 옵션:
- Bytes (B)
- Kilobytes (KB)
- Megabytes (MB)
- Gigabytes (GB)
```

## 시간 조건

### 시간 기준 종류
- 파일 생성일 : 파일을 생성한 일자 기준
- 파일 수정일 : 파일을 마지막으로 수정한 일자 기준
- 파일 열람일 : 파일을 마지막으로 열어본 일자 기준
- 파일 추가일 : 파일을 현재 폴더에 추가한 일자 기준

### 상세 조건
- **이후**: 파일의 기준일이 입력일 이후인 경우 참
- **이전**: 파일의 기준일이 입력일 이전인 경우 참
- **특정일**: 파일의 기준일이 입력일과 동일한 경우 참
- **기간**: 파일의 기준일이 입력 기간 이내인 경우 참
- **요일**: 파일의 기준일의 요일이 입력 요일과 동일한 경우 참
- **최근**: 파일의 기준일이 현재 시간에서 입력 시간 이내인 경우 참
  - 예: 현재일 2025-09-30 기준 최근 15일 이내 = 2025-09-16 ~ 2025-09-30
- **주기**: 파일의 기준일이 현재 시간에서 입력 시간 전부터인 경우 참
  - 예: 현재일 2025-09-30 기준 15일 전부터 = 2025-09-16 ~ 2025-09-30

## 태그 조건

- 파일이 해당하는 태그(이름 + 색상)를 가지고 있는지

## 모든 파일

- 폴더 내의 모든 파일에 대해서 액션을 수행

## 조건 조합

하나의 조건에 들어가는 요소들은 **OR**로, 분리된 조건들은 **AND**로 작동합니다.

### 예시

```

조건 1: 확장자 = jpg, png, docx
조건 2: 이름에 "스크린샷" 단어를 포함

```
위의 경우 조건 1에 있는 jpg **OR** png **OR** docx 확장자를 가진 "스크린샷" 단어를 포함하는 파일에 대해서 액션이 수행됩니다.

### NOT 조건
조건 앞에 있는 체크박스를 선택해 박스가 빨간색으로 변하면 해당 규칙은 **NOT** 조건이 부여되어 반대로 적용됩니다.
