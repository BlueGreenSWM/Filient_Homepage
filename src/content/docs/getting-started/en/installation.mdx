# Installation and Setup

Step-by-step guide to get started with Filient.

## 1. Installing Filient

### Download
Download the latest version from the [Filient official website](https://filient.ai)

### Installation Process
1. Open the downloaded `Filient.dmg` file
2. Drag the Filient app to the Applications folder
3. Launch Filient from the Applications folder

## 2. Initial Setup

### Permission Settings

Filient requires macOS system permissions to manage files.

**Full Disk Access**
1. When you launch Filient, a permission request popup will appear
2. Click "Open System Preferences"
3. **System Preferences > Security & Privacy > Privacy > Full Disk Access**
4. Click the lock icon in the bottom left and enter your password
5. Check the Filient checkbox
6. Restart the Filient app

> 💡 **Tip**: Without granting permissions, access to some folders may be restricted, limiting functionality.

### Language Settings
1. Click the **Settings** icon (⚙️) at the bottom of the left sidebar
2. In the **General** section, find **Language**
3. Click on **Language** to open language options
4. Select your preferred language:
   - **English** - English interface
   - **한국어** - Korean interface
5. The app will automatically refresh with the selected language

![Language Settings](/docs/images/language-settings.png)

> 💡 **Tip**: Language changes apply immediately without restarting the app.

## 3. Adding Your First Folder

### How to Add Folders

**Method 1: Using the Folder Selector**
1. Click the **"Add Folder"** button in the sidebar
2. Select the folder you want to manage in the Finder window
3. Click the **"Select"** button

**Method 2: Drag and Drop**
1. Select a folder in Finder
2. Drag and drop it into the Filient sidebar

## Troubleshooting

### When the app won't launch
1. Check macOS version (10.15 or later required)
2. Reinstall the app
3. Restart your system

### When permission errors occur
1. Recheck permissions in System Preferences
2. Completely quit and restart the Filient app
3. Restart macOS and try again
