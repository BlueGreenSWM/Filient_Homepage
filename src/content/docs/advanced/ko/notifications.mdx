# 알림 및 승인

Filient는 파일 작업에 대한 실시간 알림과 중요한 작업에 대한 사전 승인 기능을 제공합니다.

## 알림 시스템 개요

### 알림의 중요성

**즉각적인 피드백**
- 작업 완료 확인
- 오류 발생 인지
- 진행 상황 추적

**투명성 확보**
- 모든 작업 기록
- 변경 사항 추적
- 감사 로그 제공

**안전성 향상**
- 위험한 작업 경고
- 실수 방지
- 사전 검증

## 알림 유형

### 정보 알림
- 작업 시작/완료
- 파일 처리 현황
- 일반 상태 업데이트

### 경고 알림
- 잠재적 문제 감지
- 디스크 공간 부족
- 중복 파일 발견

### 오류 알림
- 작업 실패
- 권한 문제
- 시스템 오류

### 성공 알림
- 작업 완료
- 목표 달성
- 정리 완료

## 알림 채널 설정

### macOS 알림 센터
기본 시스템 알림 사용:
1. 시스템 환경설정 > 알림
2. Filient 앱 찾기
3. 알림 스타일 선택
4. 사운드/배지 설정

### 이메일 알림
중요한 작업 요약 전송:
1. 설정 > 알림 > 이메일
2. 이메일 주소 입력
3. 알림 빈도 설정
4. 알림 유형 선택

### 슬랙 통합
팀 협업을 위한 슬랙 연동:
1. 슬랙 웹훅 URL 획득
2. Filient 설정에 입력
3. 채널 지정
4. 알림 필터 설정

## 승인 워크플로우

### 승인이 필요한 작업

#### 대량 삭제
- 10개 이상 파일 삭제
- 1GB 이상 데이터 삭제
- 시스템 파일 삭제

#### 중요 파일 이동
- 문서 폴더 외부 이동
- 네트워크 드라이브 이동
- 외부 저장장치 이동

#### 파일명 일괄 변경
- 50개 이상 파일
- 확장자 변경
- 특수 문자 제거

### 승인 프로세스

1. **작업 분석**
   - 영향 받는 파일 목록
   - 예상 결과 미리보기
   - 위험도 평가

2. **승인 요청**
   - 알림 발송
   - 상세 정보 제공
   - 승인/거절 옵션

3. **실행 대기**
   - 사용자 응답 대기
   - 타임아웃 설정
   - 자동 거절 옵션

4. **작업 실행**
   - 승인 시 즉시 실행
   - 거절 시 취소
   - 결과 알림

## 알림 필터링

### 중요도 기반 필터
- 🔴 긴급: 즉시 알림
- 🟡 중요: 요약 알림
- 🟢 일반: 로그만 기록

### 시간대 설정
- 업무 시간: 모든 알림
- 야간: 긴급만
- 주말: 사용자 정의

### 규칙별 설정
각 규칙마다 개별 알림 설정:
- 알림 활성/비활성
- 알림 채널 선택
- 중요도 설정

## 알림 히스토리

### 알림 로그 보기
1. Filient 열기
2. 알림 > 히스토리
3. 필터/검색
4. 상세 정보 확인

### 로그 내보내기
- CSV 형식 지원
- 날짜 범위 선택
- 필터 적용 후 내보내기

## 고급 설정

### 알림 그룹화
유사한 알림 묶어서 표시:
- 같은 규칙의 알림
- 연속된 작업 알림
- 시간대별 그룹화

### 조건부 알림
특정 조건에서만 알림:
- 특정 폴더 작업
- 특정 파일 유형
- 특정 크기 이상

### 알림 우선순위
- 시스템 오류 > 사용자 작업
- 수동 실행 > 자동 실행
- 대량 작업 > 개별 작업

## 문제 해결

### 알림이 오지 않을 때
1. macOS 알림 설정 확인
2. Filient 알림 설정 확인
3. 방해금지 모드 확인
4. 권한 설정 재확인

### 너무 많은 알림
- 알림 그룹화 활성화
- 중요도 필터 조정
- 요약 모드 사용
- 규칙별 알림 조정

## 모범 사례

### 효율적인 알림 관리
- 중요한 작업만 즉시 알림
- 일반 작업은 요약으로
- 야간 시간 알림 최소화
- 정기적인 알림 로그 확인

### 승인 워크플로우 최적화
- 위험한 작업만 승인 요구
- 신뢰할 수 있는 규칙은 자동 실행
- 승인 타임아웃 적절히 설정
- 승인 히스토리 정기 검토
