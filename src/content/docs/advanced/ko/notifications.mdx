# 알림 및 승인
![알림](/docs/images/notification.png)

Filient는 파일 작업에 대한 실시간 알림과 중요한 작업에 대한 사전 승인 기능을 제공합니다.

## 알림 유형

### 정보 알림
- 작업 완료
- 파일 처리 현황
- 승인 필요 규칙

### 오류 알림
- 작업 실패
- 권한 문제
- 시스템 오류

## 네이티브 알림 설정

설정 -> 알림 설정에서 시스템 알림권한 -> 설정 확인 클릭 -> macOS 알림 센터 열기

![알림](/docs/images/notification-permission.png)

### macOS 알림 센터
기본 시스템 알림 사용:
1. 시스템 환경설정 > 알림
2. Filient 앱 찾기
3. 알림 허용

## 승인
![승인](/docs/images/approval.png)

- 승인 대기열에서 미리보기 / 승인 / 거부가 가능합니다
- 여러 항목 선택 후 일괄 승인도 가능합니다.

## 알림 목록

### 알림 필터링 및 정리
1. 알림 클릭 시 읽음 처리
2. 알림 체크박스 선택 시 읽음 / 삭제 처리 가능
3. 검색을 통해서 알림 필터링 가능

## 문제 해결

### 알림이 오지 않을 때
1. macOS 알림 설정 확인
2. Filient 알림 설정 확인
3. 방해금지 모드 확인
4. 권한 설정 재확인
