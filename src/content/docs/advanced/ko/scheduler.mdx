# 규칙 스케줄러

규칙 스케줄러는 설정한 시간에 자동으로 규칙을 실행하여 파일 정리를 완전히 자동화합니다.

## 스케줄러 개요

### 자동 실행의 이점

**시간 절약**
- 수동 실행 불필요
- 정기적인 정리 자동화
- 일관된 폴더 상태 유지

**편의성**
- 잊지 않고 실행
- 최적 시간대 선택 가능
- 무인 실행 (야간, 주말)

**효율성**
- 파일이 쌓이기 전 처리
- 시스템 리소스 여유 시간 활용
- 작업 부하 분산

## 전역 스케줄러 설정

### 실행 주기 옵션

#### 매일 실행
- 다운로드 폴더 정리
- 임시 파일 삭제
- 일일 백업

#### 매주 실행
- 주간 보고서 정리
- 프로젝트 파일 아카이브
- 대용량 파일 정리

#### 매월 실행
- 월별 사진 정리
- 오래된 문서 아카이브
- 디스크 정리

## 개별 규칙 스케줄

### 규칙별 스케줄 설정

각 규칙마다 개별 스케줄을 설정할 수 있습니다.

1. 규칙 선택
2. "스케줄 설정" 클릭
3. 실행 주기 선택
4. 시간 설정
5. 저장

### 스케줄 우선순위

- 개별 스케줄 > 전역 스케줄
- 중복 시 한 번만 실행
- 충돌 방지 로직 내장

## 고급 스케줄링

### Cron 표현식

고급 사용자를 위한 Cron 표현식 지원:

```
# 매시간 30분마다
30 * * * *

# 평일 오전 9시
0 9 * * 1-5

# 매월 1일과 15일
0 0 1,15 * *
```

### 조건부 실행

특정 조건에서만 실행:
- 파일 개수가 특정 수 이상
- 폴더 크기가 임계값 초과
- 특정 파일 존재 시

### 스케줄 그룹

관련 규칙을 그룹으로 묶어 동시 실행:

1. 스케줄 그룹 생성
2. 규칙들 추가
3. 그룹 스케줄 설정
4. 일괄 실행

## 모니터링과 로그

### 실행 히스토리

- 실행 시간
- 처리된 파일 수
- 성공/실패 상태
- 소요 시간

### 알림 설정

- 실행 시작 알림
- 완료 알림
- 오류 알림
- 요약 리포트

## 문제 해결

### 스케줄이 실행되지 않을 때

1. Filient가 백그라운드에서 실행 중인지 확인
2. Mac의 절전 설정 확인
3. 스케줄 활성화 여부 확인
4. 시스템 시간 설정 확인

### 성능 최적화

- 야간 시간대 활용
- 규칙 우선순위 조정
- 동시 실행 제한
- 리소스 모니터링
