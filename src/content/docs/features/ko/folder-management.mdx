# 폴더 관리

폴더 관리는 Filient의 핵심 기능으로, 자동 정리할 폴더를 추가하고 관리하는 기능입니다.

## 폴더 추가하기

### 폴더 선택기
1. 사이드바 상단의 ```+``` 버튼 클릭
2. Finder 대화상자에서 원하는 폴더 선택
3. **"선택"** 클릭

## 폴더 구성

### 폴더 정보 보기
폴더를 선택하면 다음 정보를 확인할 수 있습니다:
- **폴더 경로**: 전체 경로
- **즐겨찾기 여부**: 해당 폴더가 즐겨찾기가 되어있는지 여부

### 즐겨찾기 설정
자주 사용하는 폴더를 즐겨찾기로 설정:
1. 폴더 선택 or 사이드바 폴더 목록 편집 선택
2. 폴더명 우측 ⭐ 아이콘 클릭
3. 사이드바 상단에 고정됨

## 계층 구조 관리

### 하위 폴더 표시
- 상위 폴더에 하위 폴더 포함 규칙이 존재하는 경우에만 표시
- 폴더 왼쪽 ▶ 아이콘 클릭으로 펼치기/접기
- 하위 폴더는 들여쓰기로 표시

### 하위 폴더 규칙 적용
1. 상위 폴더 선택
2. 규칙 생성 시 "하위 폴더 포함" 체크
3. 모든 하위 폴더에 자동 적용
