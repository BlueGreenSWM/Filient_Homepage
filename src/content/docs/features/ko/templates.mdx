# 규칙 템플릿

템플릿을 사용하면 자주 사용하는 규칙을 빠르게 설정할 수 있습니다.

## 템플릿 사용하기

1. **템플릿 라이브러리 열기**
   - 폴더 선택 > 새 규칙 > "템플릿에서 가져오기"

2. **템플릿 선택**

3. **필요시 수정**
   - 규칙 이름 및 설명 수정
   - 폴더 경로 설정
   - 조건 값 수정
   - 액션 옵션 변경 등

4. **저장 및 적용**
   - 저장 후 활성화

## 커스텀 템플릿 만들기

### 템플릿으로 저장

#### 규칙 생성 페이지
1. 기존 규칙 선택 또는 새 규칙 생성
2. 하단의 "템플릿으로 저장" 옵션 활성화

#### 규칙 템플릿 페이지
1. "새 템플릿 만들기" 버튼 선택
2. 템플릿 이름, 설명, 조건, 액션 등 정보 입력 후 하단의 저장 선택

### 템플릿 관리
템플릿 우측의 점 3개를 선택하면 메뉴를 확인할 수 있습니다.
- **편집**: 템플릿 수정
- **삭제**: 불필요한 템플릿 제거
