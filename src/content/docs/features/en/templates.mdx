# Rule Templates

Templates allow you to quickly set up frequently used rules.

## Using Templates

1. **Open Template Library**
   - Select folder > New Rule > "Import from Template"

2. **Select Template**

3. **Modify if Needed**
   - Edit rule name and description
   - Set folder path
   - Modify condition values
   - Change action options, etc.

4. **Save and Apply**
   - Save and activate

## Creating Custom Templates

### Save as Template

#### Rule Creation Page
1. Select an existing rule or create a new rule
2. Enable the "Save as Template" option at the bottom

#### Rule Template Page
1. Select "Create New Template" button
2. Enter template information (name, description, conditions, actions, etc.) and click save at the bottom

### Manage Templates
Select the three dots to the right of a template to see the menu:
- **Edit**: Modify template
- **Delete**: Remove unnecessary templates
